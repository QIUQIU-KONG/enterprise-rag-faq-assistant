# 申报材料清单与填报要求

> **领域**: 项目申报材料
> **子主题**: 各项目申报所需材料清单及系统填报注意事项


## 一、未来工厂 — 材料清单

### 智能工厂申报材料
1. 智能制造能力成熟度自评估报告(在"浙企智造在线"平台在线完成)
2. 企业基本信息(统一信用代码、法人、联系人等)
3. 企业数字化转型情况说明
4. 智能工厂建设方案(含目标、技术路线、实施计划、预期成效)
5. 推荐企业汇总表(各市经信局统一报送)

### 先进/卓越/领航智能工厂
1. 智能工厂常态化管理台账
2. 企业出海业务证明(如适用，用于"领航"申报)
3. 海外工厂/项目推荐表(如适用)
4. 动态评估相关数据

### 未来工厂试点
1. 未来工厂培育项目申报书
2. 企业数字化/网络化/智能化现状分析报告
3. 建设方案(含投资预算、实施周期、考核指标)
4. 相关证明材料(营业执照、知识产权、资质证书、财务报表)

## 二、智算云企 — 材料清单

### 智算云企申报
1. 企业营业执照(在浙江省内注册)
2. 企业基本情况介绍(含智算业务描述)
3. 智算基础设施情况(服务器规模、算力容量、网络带宽等)
4. 专业技术团队证明(人员数量、学历/职称结构、核心成果)
5. 近两年财务报表或审计报告
6. 知识产权证明(专利、软著、论文等)
7. 企业信用报告(无违法违规和严重失信记录)

### 智算云应用场景申报
1. 应用场景说明报告(含创新性、示范性和可复制性分析)
2. 项目权属证明(知识产权归属清晰)
3. 申报单位营业执照或法人证书
4. 同意公开发布的声明(申报即视为同意)

### 填报注意事项
- 申报平台: https://zj87.jxt.zj.gov.cn/
- 登录方式: 浙江政务服务网法人账号或企业码扫码
- **不需要纸质材料**，全程在线完成
- 附件命名规范: "序号_材料名称_企业名称.pdf"
- 单个文件不超过10MB

## 三、雄鹰企业 — 材料清单

### 通用材料
1. 雄鹰企业申报表(在线填写)
2. 企业营业执照
3. 近三年审计报告(或财务报表)
4. 企业简介(含主营业务、行业地位、核心竞争力)
5. 知识产权清单(专利、商标、软著等)
6. 研发投入证明(研发费用专项审计或纳税申报表)
7. 社会信用报告(无严重失信记录)

### 链主企业附加材料
1. 产业链上下游合作关系证明(核心供应商/客户清单及合作协议)
2. 行业引领带动作用说明(对产业链的技术/标准/市场影响)
3. 关键技术突破或进口替代证明

### 制造业总部领军企业附加材料
1. 总部管理职能说明(财务/人事/研发/战略的集中管控证明)
2. 在浙子公司/分支机构清单
3. 税收贡献证明(在浙纳税情况)

### 申报领域与主管部门对应
| 领域 | 主管部门 |
|------|----------|
| 现代农业 | 省农业农村厅 |
| 工业/数字经济 | 省经信厅 |
| 现代服务业 | 省发改委 |
| 贸易/建筑 | 省国资委 |

## 四、新材料首批次 — 材料清单

1. 《重点新材料首批次应用示范指导目录修订建议汇总表》(Excel版)

## 通用填报规范

1. **浏览器**: 使用Chrome或Edge最新版，不要用360浏览器兼容模式
2. **保存草稿**: 每10分钟保存一次，系统有30分钟自动退出机制
3. **数据单位**: 仔细区分"万元"与"元"，金额大写用"叁佰万元整"
4. **日期格式**: YYYY-MM-DD
5. **附件格式**: PDF优先，扫描分辨率300dpi以上，公章清晰
6. **纸质材料**: A4双面打印，左侧胶装，封面按统一模板，盖骑缝章
