# 浙江省重点企业项目申报概览

> **领域**: 项目申报材料
> **子主题**: 2026年度浙江省重点企业项目类型、条件和适用范围


## 当前可申报的四个重点项目

### 一、未来工厂梯度培育

- **发文单位**: 浙江省经济和信息化厅
- **依据文件**: 《浙江省制造业数字化转型行动方案》(浙制造发〔2024〕34号)、《未来工厂梯度培育实施指南(试行)》(浙经信产数〔2025〕99号)
- **适用企业**: 制造业企业，数字化水平2.0以上
- **三个层级**:
  1. **智能工厂** — 基础级，需在"浙企智造在线"平台完成自评估和申报，截止日期5月10日
  2. **先进/卓越级智能工厂** — 需常态化管理和动态评估，有出海业务的企业可自愿申报"领航智能工厂"
  3. **未来工厂** — 最高层级，省级未来工厂培育项目库和试点企业
- **关键平台**: "浙企智造在线"(浙江省企业之家网)

### 二、智算云企

- **发文单位**: 浙江省经济和信息化厅
- **文件编号**: 浙经信设施函〔2026〕73号
- **依据文件**: 《关于促进智算创新发展实施方案(2025-2027年)》(浙制高办〔2025〕38号)
- **两类申报**:
  1. **智算云企** — 从事智算服务的企业，需在浙江注册、独立法人、无不良记录，具备智算硬件设施和专业人才
  2. **智算云应用场景** — 企事业单位均可，应用场景需具创新性、示范性和可推广性
- **截止日期**: 2026年6月25日
- **申报平台**: https://zj87.jxt.zj.gov.cn/ (浙江省企业之家网)
- **登录方式**: 浙江政务服务网法人账号或企业码扫码

### 三、雄鹰企业(链主/制造业总部领军企业)

- **发文单位**: 浙江省制造业高质量发展(数字经济发展)领导小组办公室
- **文件编号**: 浙制高办〔2026〕32号
- **依据文件**: 《2026年全省制造业高质量发展实施"雄鹰行动"工作要点》(浙制高办〔2026〕11号)
- **四个申报领域**:
  1. 现代农业 — 国家级农业龙头企业，省农业农村厅负责遴选
  2. 工业/数字经济 — 省经信厅负责遴选，含制造业总部领军企业
  3. 现代服务业 — 科技/物流/金融/环保/文旅等，省发改委负责遴选
  4. 贸易/建筑/跨国 — 省属企业/子公司，省国资委负责遴选
- **截止日期**: 2026年7月10日
- **申报方式**: 在线申报

### 四、重点新材料首批次应用示范

- **发文单位**: 工业和信息化部原材料工业司 → 浙江省经信厅转发(浙经信材料便函〔2026〕59号)
- **内容**: 征集《重点新材料首批次应用示范指导目录》修订建议
- **截止日期**: 2026年6月10日前报省经信厅材料处
- **联系人**: 省经信厅材料处 0571-87058139
- **要求**: 填写修订建议汇总表，报送电子版

---

## 通用申报条件

1. 在浙江省内注册的独立法人企业
2. 正常经营，无重大违法违规和严重失信记录
3. 具备与申报方向相匹配的研发能力、人才团队和硬件条件
4. 知识产权归属清晰，无产权纠纷
