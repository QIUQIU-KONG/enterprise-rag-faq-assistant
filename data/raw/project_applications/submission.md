# 项目申报提交流程

> **领域**: 项目申报材料
> **子主题**: 各项目的完整申报流程和关键时间节点


## 一、未来工厂梯度培育 — 申报流程

### 智能工厂申报流程
1. 企业登录"浙企智造在线"平台(浙江省企业之家网)
2. 完成"智能制造能力成熟度自评估"(需达到数字化水平2.0以上)
3. 参照《智能工厂梯度培育要素条件(2025年版)》在线填报
4. 填报完成后在线提交，无需纸质材料
5. **申报截止: 2026年5月10日**前将推荐企业汇总表报送省经信厅

### 先进/卓越级智能工厂
1. 由各市经信局组织推荐
2. 企业自愿申报，需常态化管理和动态评估
3. 有出海业务的企业可同步申报"领航智能工厂"
4. **申请截止: 2026年3月5日**前将企业和海外项目推荐表报送省经信厅

### 未来工厂
1. 根据《浙江省未来工厂分级培育建设要求(试行)》
2. 企业提交未来工厂培育项目申报
3. 省经信厅组织评审，纳入省级项目库
4. 试点企业按建设方案推进，接受中期检查和验收

## 二、智算云企 — 申报流程

### 申报步骤
1. **网上填报**: 登录浙江政务服务网(https://zj87.jxt.zj.gov.cn/)
2. 选择"2026年度浙江省智算云企申报"项目
3. 使用浙江政务服务网法人账号或企业码(法人/经办人)扫码登录
4. 按系统提示逐项填写企业基本信息和申报内容
5. 上传所需附件材料(营业执照、财务报表、知识产权证明等)
6. 确认无误后在线提交，无需纸质材料
7. **申报截止: 2026年6月25日**

### 各市推荐流程
- 各市经信局按照"自愿申报、择优推荐"原则
- 统筹组织辖区内企业申报
- 审核后择优推荐上报省经信厅

## 三、雄鹰企业 — 申报流程

### 雄鹰企业遴选流程
1. **企业申报**: 7月10日前在指定平台在线申报
2. **领域分工**:
   - 现代农业 → 省农业农村厅组织遴选
   - 工业/数字经济 → 省经信厅组织遴选(含制造业总部领军企业)
   - 现代服务业 → 省发改委组织遴选
   - 贸易/建筑领域 → 省国资委组织遴选(含省属企业及子公司)
3. 各主管部门审核推荐
4. 省领导小组办公室汇总评审
5. 公示通过后正式认定

### 链主企业遴选
- 在雄鹰企业评选基础上进行
- 重点考察企业在产业链中的引领带动作用
- 需提交额外的产业链上下游合作证明材料

## 四、新材料首批次 — 申报流程

### 修订建议提交流程
1. 各市经信局将通知转发至辖区内县(市、区)
2. 广泛动员相关企业研究并提出目录修订建议
3. 企业填写《重点新材料首批次应用示范指导目录修订建议汇总表》
4. 各市汇总整理、审核确认后上报
5. **截止时间: 2026年6月10日**前报省经信厅材料处
6. 需同时报送电子版文件和盖章扫描件
