# 申报常见问题与避坑指南

> **领域**: 项目申报材料
> **子主题**: 项目申报过程中最常遇到的问题及解决方案


## 一、申报资格问题

### Q1: 企业不符合基本条件怎么办？
- 核查营业执照经营范围是否覆盖申报方向，如有缺口先办理经营范围变更
- 检查企业信用状态(通过"信用浙江"或"国家企业信用信息公示系统")
- 确认近三年无重大违法违规记录
- 涉外企业需确认知识产权权属清晰，无跨国产权纠纷

### Q2: 数字化水平不够2.0可以申报智能工厂吗？
- 不可以。需先在"浙企智造在线"平台完成自评估，达到2.0以上才具备申报资格
- 建议先推进企业内部的数字化改造，达标后再申报下一批次

### Q3: 同一企业可以同时申报多个项目吗？
- 可以。不同项目彼此独立，但需分别准备材料、分别提交
- 注意各项目截止日期不同，合理安排先后顺序

---

## 二、时间节点问题

### Q4: 截止日期错过了还能补报吗？
- **不可以**。所有项目的截止日期均为刚性要求
- 建议提前至少2周完成材料准备，提前3天完成系统提交
- 各项目截止日期：智能工厂5/10、智算云企6/25、雄鹰7/10、新材料6/10

### Q5: 系统临近截止日期提交失败怎么办？
- 截止日期前系统访问量巨大，可能拥堵
- **必须提前提交**，不要等到最后一天
- 如遇系统故障，截屏留证并立即联系主管部门

---

## 三、材料准备问题

### Q6: 营业执照范围不包含申报方向
这是最常见的退回原因之一。解决方案：
1. 提前核查经营范围
2. 如不覆盖，先到市场监管局办理经营范围增项
3. 增项完成后再提交申报

### Q7: 财务报表数据与税务申报不一致
- 申报前核对三表一致: 审计报告 + 纳税申报表 + 财务报表
- 如存在差异，需准备书面说明解释原因
- 自筹资金来源必须明确(银行存款证明或董事会出资决议)

### Q8: 知识产权过期或权属不清
- 检查所有专利年费缴纳状态(逾期未缴的专利无效)
- 确认知识产权权属(无共有纠纷、无质押争议)
- 联合申报项目需提前与合作单位签订正式合作协议(双方公章)

## 四、系统填报问题

### Q9: 浏览器兼容性导致页面显示异常
- **严禁使用**: IE浏览器、360浏览器兼容模式
- **推荐使用**: Chrome最新版或Edge最新版
- 如遇格式错乱，清空浏览器缓存后重试

### Q10: 填写过程中被系统踢出
- 系统有30分钟无操作自动退出机制
- **每10分钟点击一次"保存草稿"按钮**
- 展开所有折叠区域检查是否有隐藏必填项
- 全部填完后先预览再提交

### Q11: 附件上传后不要忘记检查
- 上传后逐个打开附件确认内容正确、方向正常
- 单个文件不超过10MB
- 同类多页材料合并为单个PDF，不得拆分为多个文件上传

## 五、常见被退回原因

| 退回原因 | 如何避免 |
|----------|----------|
| 材料不全(漏盖章/漏签字) | 建立Checklist逐项勾对，至少2人交叉审核 |
| 承诺书格式不符 | 使用申报通知中的最新模板(不要用去年的) |
| 资质证书过期 | 逐项检查有效期(高企证书3年有效、ISO认证3年) |
| 数据前后矛盾 | 修改一个数据后，检查所有引用了该数据的表格 |
| 超过补充材料时限 | 收到退回通知当天立即确认，排期处理 |
| 预算各科目比例超限 | 逐科目对照申报指南中的比例上限 |

---

## 六、终极建议

1. **早启动**: 通知发布当天就开始，不要等到截止日期前
2. **建清单**: 把所有材料列成Excel checklist，完成一项勾一项
3. **多审校**: 至少两个人交叉审核(一个懂技术、一个懂财务)
4. **存版本**: 每次修改保留旧版本，用日期命名
5. **备预案**: 系统提交可能出问题，预留至少3天buffer
6. **主动沟通**: 不确定的事项电话咨询主管部门经办人，不要自己猜
